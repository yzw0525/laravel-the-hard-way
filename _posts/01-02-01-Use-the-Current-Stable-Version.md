## 在 github 上添加 SSH key 的步骤{#use_the_current_stable_version_title}

首先需要检查你本地是否已经有 [SSH key]
 
{% highlight console %}
> la -al ~/.ssh
{% endhighlight %}



进入 id_rsa.pub 目录粘贴密匙到 github账户的 点击菜单栏的 SSH key 进入页面添加Add SSH Key中
