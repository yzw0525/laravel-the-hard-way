# Laravel doc for SK-Media

[在线地址](http://sk-media.github.io/laravel-the-hard-way/)

## 概览

这是 _Laravel doc_ 项目的 GitHub Pages 仓库。

* 这个网站是由 Jekyll 构建的。
* 每个区块以及子区块都是位于 `_posts/` 目录下的 markdown 文件。
* 子区块可以在配置中设定 `isChild: true` 。
* 导航以及页面结构都是自动生成的。


## 如何贡献

1. Fork 然后编辑
2. 安装 [Ruby](https://rvm.io/rvm/install/) 和 [Jekyll](https://github.com/mojombo/jekyll/) 在本地预览
3. 提交一个 pull request

### 贡献者样式规范

1. 使用美式英语拼写 (*仅适用于英语的 repo*)
2. 使用 4 个空格来缩进，不要使用 tabs。
3. 每隔 120 个字符换行。

## 为什么会有这个 repo

Laravel Framework 是我们需要新学习的一个全栈 Web 框架，这里是大家一起总结、积累的地方。
